GoSy Run
================
Helps you exercise running.

Features
----------------

### Interval timer
Configurable with extended rest, warm up and cool down

### Ladder timer
Ascending, Descending, Asc-Desc and Desc-Asc

### Stretch timer
With 8 stretch types
