GoSy Run
================
Helps you exercise running.

Features
----------------

### Interval timer
Configurable with extended rest, warm up and cool down

### Ladder timer
Ascending, Descending, Ascending-Descending and Descending-Ascending

### Stretch timer
With 8 stretch types
